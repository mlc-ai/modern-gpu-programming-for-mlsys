
.. _chap_appendix:

Appendix
========


The main tutorial path ends with Writing TIRX Kernels with Agents. The
appendix is for setup, API lookup, optional practice kernels, and source
references.

**How to use this appendix.**

+--------------------------------------+-------------------------------+
| Need                                 | Use                           |
+======================================+===============================+
| Install TVM/TIRX or run a tiny       | **Environment Setup**         |
| kernel                               |                               |
+--------------------------------------+-------------------------------+
| Look up the spelling of a TIRX API   | **TIRX API Reference**        |
| used in the tutorial                 |                               |
+--------------------------------------+-------------------------------+
| Understand parsing, buffers, scopes, | **TIRX Language and Compile   |
| generated CUDA, or the compile       | Pipeline**                    |
| pipeline                             |                               |
+--------------------------------------+-------------------------------+
| Practice basic thread indexing,      | **Practice Kernel: Fused GELU |
| elementwise code, and reductions     | Gate** and **Practice Kernel: |
|                                      | RMSNorm Reduction**           |
+--------------------------------------+-------------------------------+
| Read the full Flash Attention 4      | **Flash Attention 4 Source**  |
| kernel source                        |                               |
+--------------------------------------+-------------------------------+

Most readers only need **Environment Setup** and **TIRX API Reference**
at first. The practice kernels are optional.
